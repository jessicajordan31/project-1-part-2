# Her Capital - Jess Jordan, Ella Jones, Preeyanti Sen
## DePauw CSC340 Project I Part II
### Description
This project is the frontend for a financial advising website for women. It uses the Vue framework, HTML/CSS, and JS for the frontend. The pages that we added were Home, About, Connect, Support, Login, and Dashboard.

### Git Commands
- After confirming to be a contributor, open a new file on your computer and write: `git clone https://github.com/jessicajordan31/project-1-part-2.git`
- When you have edited a slide and are ready to send it to this repo, write: ` git add . ` to add your updates to the staging area
- Then, to commit it to the file, write: ` git commit -m "message" ` to save your work with a message
- Lastly, push it to this repo by writing `git push origin main`
- To see / update changes that others have made (always do this before starting to write your code!), write: `git pull `

### Vue Locations
- Add images to \src\assets
- Add pages to  \src\components
